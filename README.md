# SRTA3353 — Smart Security IoT System

Group project for SRTA3353 Machine Learning for IoT, Academic Session 2025/2026.

## Project Overview
Real-time danger sound detection and human presence monitoring using 
ESP32-CAM, INMP441 microphone, and HC-SR501 PIR sensor.

## Tech Stack
- Hardware: ESP32-CAM, INMP441, HC-SR501, Active Buzzer
- ML: YOLOv6-nano (human detection), MobileNetV2 (audio classification)
- Dashboard: Favoriot
- Languages: Python, C++ (Arduino)
